# Project Overview
This repository contains a Clinical Visits Overview Dashboard designed to explore and visualize important insights from healthcare visit data. The dashboard highlights key patient metrics such as demographics, wait times, department usage, and disease patterns. It is ideal for showcasing skills in data analysis and interactive visualization perfect for a portfolio or data science presentation.

## Features

 - Visual breakdown of patient demographics (gender, age, race)
 - Interactive view of departments and service usage
 - Analysis of patient wait times and averages
 - Disease and wait time correlation chart
 - Time-based filtering (by quarter/year)
 - Clear, intuitive charts and layouts

## Dataset

- The dataset contains anonymized clinical visit records, including:
- Patient demographics (gender, race, age)
- Visit department information
- Wait times
- Disease classifications
- Time period (quarters/years)
